값이 할당되어 있는 변수에 다시 값을 할당하는 것을 말한다. 자바스크립트에서는 [var](var.md), [let](Javascript/let.md)으로 선언된 변수만이 재할당이 허용된다. 

자바스크립트의 변수 재할당은 특이하게도, 변수(즉, [식별자](식별자.md))가 가리키는 메모리 공간에 저장되어 있는 값을 바꾸는 것이 아니라 **새로운 값에 맞는 새로운 메모리 공간에 재할당 값을 저장한 뒤 [식별자](식별자.md)가 가리키는 메모리 공간을 변경시킨다.** 이전 값에 사용된 메모리 공간은 추후 가비지 컬렉터에 의해 자동으로 메모리에서 해제된다.
