[Kotlin](Kotlin.md)에서 `is`는 `instanceOf`와 같은 기능으로, 변수의 타입을 검사할 뿐만 아니라 타입이 일치하는 경우 컴파일러가 그 타입으로 변환을 해준다. 이러한 기능을 스마트 캐스트라고 한다.

```kotlin
interface Expr
class Num(val value: Int): Expr
class Sum(val left: Expr, val right: Expr): Expr

// 스마트 캐스르로 인해 명시적 캐스트 없이 프로퍼티에 접근이 가능하다.
fun eval(e: Expr): Int =
    if (e is Num) {
        e.val
    } else if (e is Sum) {
        eval(e.right) + eval(e.left)
    } else {
        throw IllegalArgumentException("Unknown expression")
    }
    
// 보다 코틀린스럽게 작성하려면 다음과 같이 작성할 수 있다.
fun eval(e: Expr): Int = 
    when (e) {
        is Num -> e.value
        is Sum -> eval(e.right) + eval(e.left)
        else -> throw IllegalArgumentException("Unknown expression")
    }
```

스마트 캐스트는 `is`로 변수에 든 값의 타입을 검사한 다음에 **그 값이 바뀔 수 없는 경우**에만 작동한다. 왜냐하면 `val`이 아니거나 `val`이지만 커스텀 접근자를 사용하는 경우에는 해당 프로퍼티에 대한 접근이 항상 같은 값을 내놓는다고 확신할 수 없기 때문이다.

이러한 경우에 원하는 타입으로 명시적으로 타입 캐스팅을 하려면 `as` 키워드를 사용하면 된다.

```kotlin
val n = e as Num
```
