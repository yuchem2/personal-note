```kotlin
fun main() {
    val input = redln()
    val name = if (input.isNotBlank()) input else "Kotlin"
    println("Hello, $name!")
}
```

문자열 내부에 `$`와 함께 변수 이름을 넣으면 변수를 문자열 안에 참조할 수 있다. 만약 `$`를 넣고 싶다면 `\`를 사용해 이스케이프시켜야 한다. 

만약 템플릿 안에 복잡한 식을 사용할려면 `{}`안에 식을 넣어야 한다.

```kotlin
fun main() {
    val name = readln()
    if (name.isNotBlank()) {
        println("Hello, ${name.length}-letter person!")
    }
}
```

> **한글을 문자열 템플릿에서 사용할 경우 주의할 점**
> 코틀린에서는 모든 유니코드 문자를 변수 이름으로 사용할 수 있다. 그러므로 변수 이름을 한글로 사용할 수 있다. 그러나 문자열 템플릿에서 변수명 바로 뒤에 문자를 붙이는 경우 붙어있는 모든 문자를 식별자로 판단하여 `unresolved reference` 오류를 발생시킨다. 
> 
> 해당 오류를 해결하기 위해서는 변수 명을 `{}`로 묶어서 사용하면 된다. `${name}님 반가워요!`
