```kotlin
val str: String? = null
// 아래 두 연산은 결과가 동일하다.
if (str != null) str.uppercase() else null
str?.uppercase()
```

[널이 될 수 있는 타입](널이%20될%20수%20있는%20타입.md)을 호출할 때 `null` 검사와 메서드 호출을 한 연산으로 수행하는 연산자이다. 호출하려는 값이 `null`이 아니라면 `?.`는 일반 메서드 호출처럼 작동한다. 호출하려는 값이 `null`이면 이 호출은 무시되고 결과값은 `null`이 된다.
- `?.` 연산자의 결과 타입도 [널이 될 수 있는 타입](널이%20될%20수%20있는%20타입.md)이다.
