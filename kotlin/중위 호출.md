중위 호출은 수신 객체 뒤에 메서드 이름을 위치시키고 그 뒤에 유일한 메서드 인자를 넣는다. 이때 다른 구분자는 필요 없고 각각을 공백으로 분리한다.

```kotlin
1.to("one")
1 to "one"
```

인자가 하나뿐인 일반 [함수](함수.md)나 인자가 하나뿐인 [확장 함수](클래스%20확장.md)에만 중위 호출을 사용할 수 있다. 함수를 중위 호출에 사용하고 허용하고 싶으면 `infix` 변경자를 함수 선언 앞에 추가해야 한다.

```kotlin
// 간략화된 to 함수 정의
infix fun Any.to(other: Any) = Pair(this, other)
```
