## 변수 선언
변수 선언은 키워드(`val` 혹은 `var`)와 변수 이름, 타입 정보를 순서대로 작성하여 선언할 수 있다.

```kotlin
val question: String = "삶, 우주, 그리고 모든 것에 대한 궁극적인 질문"
val answer: Int = 42
```

여기서 타입을 지정하지 않으면 컴파일러가 초기화 식을 분석해 초기화 식의 타입을 변수 타입으로 지정한다([타입추론](Kotlin.md#타입%20추론)). 변수를 선언하면서 초기화하지 않으면 타입 추론이 불가능하기 때문에 타입을 반드시 지정해야 한다.
### 변수 선언 키워드
- `val`: 읽기 전용 참조. `val`로 선언된 변수는 단 한 번만 대입될 수 있다.
- `var`: 재대입 가능한 참조. 초기화가 이뤄진 다음이라도 다른 값을 대입할 수 있다.

기본적으로 [Kotlin](Kotlin.md)에서는 모든 변수를 `val` 키워드를 사용해 선언하는 방식을 지켜야 한다. 반드시 필요할 때만 `var` 키워드를 사용한다.

`val` 키워드를 사용하더라도 아래 코드는 정상 빌드된다. 컴파일러가 잠재적인 2가지 대입 중 단 하나만 실행될 수 있다는 것을 알고 있기 때문이다.

```kotlin
fun canPerformOption(): Boolean {
    return true
}

fun main() {
    val result: String
    if (canPerformOption()) {
        result = "Success"
    }
    else {
        result = "Can't perform operation"
    }
}
```

`val` 참조 자체가 읽기 전용이지만, 참조가 바뀌지 않는 한 참조가 가리키는 객체의 내부 값은 변경될 수 있다.

```kotlin
fun main() {
    val languages = mutableListOf("Java")
    languages.add("Kotlin")
}
```

`var` 키워드를 사용하면 변수의 값을 변경할 수 있지만 변수의 타입은 고정된다. 

```kotlin
fun main() {
    val answer = 42
    answer = "no answer" // Error: type mismatch
}
```
