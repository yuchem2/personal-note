일반적으로 람다식을 호출하는데 사용되는 함수로 `run`, `with`, `let`, `also`, `apply`가 있다. 이 중 `run`, `with`, `apply`는 수신 객체 함수이고, `let`, `also`는 수신 객체를 `it`으로 받는 일반 람다를 사용한다.

| 함수                | 참조 방법  | 반환값    |
| ----------------- | ------ | ------ |
| `x.let { ... }`   | `it`   | 람다의 결과 |
| `x.also { ... }`  | `it`   | x      |
| `x.apply { ... }` | `this` | x      |
| `x.run { ... }`   | `this` | 람다의 결과 |
| `with(x) { ... }` | `this` | 람다의 결과 |
- `null`이 아닌 경우에만 실행하고 싶으면 `let`을 [안전한 호출 연산자](안전한%20호출%20연산자.md)와 함께 사용
- 어떤 식의 결과를 변수에 담되 그 영역을 한정시키고 싶을 때 `let`을 독립적으로 사용
- 빌더 스타일의 API를 사용해 객ㄱ체 프로퍼티를 설정할 때는 주로 `apply` 사용
- 어떤 동작을 실행한 후 원래의 객체를 다른 연산에 사용하고 싶을 때는 `also` 사용
- 하나의 객체에 대해 이름을 반복하지 않으면서 여러 함수 호출을 그룹으로 묶고 싶으면 `with` 사용
- 객체를 설정한 다음에 별도의 결과를 돌려주고 싶을 때 `run` 사용
