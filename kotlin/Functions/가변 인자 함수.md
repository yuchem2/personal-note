리스트 [Collections](Collections.md)을 생성하는 함수(`listOf<T>`)는 원하는 만큼 많이 원소를 전달할 수 있다. 

```kotlin
fun listOf<T>(vararg values: T): List<T> { /* 구현 */}
```

자바에서는 가변 인자를 표현할 때 타입 뒤에 `...`을 붙이지만 코틀린에서는 파라미터 앞에 `vararg` 변경자를 붙여 표시한다. 

```kotlin
fun main(args: Array<strings>) {
    val list = listOf("args: ", *args) // 스프레드 연산자: *
    println(list)
}
```

이미 배열에 들어있는 원소를 가변 길이 인자로 넘길 때 자바에서는 그냥 넘기면 되지만 코틀린에서는 배열을 명시적으로 풀어 배열의 각 원소가 인자로 전달되게 해야 한다.
