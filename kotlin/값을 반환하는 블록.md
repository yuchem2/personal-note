`if`와 [when](when.md) 모두 분기에 블록을 사용할 수 있다. 그런 경우 블록의 마지막 문장이 블록 전체의 결과가 된다.

```kotlin
fun eval(e: Expr): Int = 
    when (e) {
        is Num -> {
            println("num: ${e.value}")
            e.value // 마지막 값이 반환된다.
        }
        is Sum -> {
            val left = eval(e.left)
            val right = eval(e.right)
            println("sum: ${left} + ${right}")
            left + right // 마지막 값이 반환된다.
        }
        else -> throw IllegalArgumentException("Unknown expression")
    }
```

**블록의 마지막 식이 블록의 결과라는 규칙은 블록이 값을 만들어내야하는 경우 항상 성립한다.** [try-catch](try-catch.md)구문, 람다 식 등에서 활용된다. 하지만 일반적인 [함수](함수.md)에서는 성립하지 않는다.
